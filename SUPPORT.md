# Support

KathaSutra is an early-stage community project. There is no guaranteed support SLA.

Use GitHub issues for reproducible bugs and scoped feature requests. Before filing:

1. search existing issues;
2. confirm the problem against the latest supported commit or release;
3. remove secrets, credentials, cookies, private source responses, personal data, and tenant data;
4. include deployment mode, database backend, relevant configuration with secrets redacted, logs, and reproduction steps.

Use the security process in `SECURITY.md` for vulnerabilities. Do not file public security issues.

Questions about third-party websites, copyright, access permissions, or source legality may be closed when the project cannot safely or lawfully assist. KathaSutra does not guarantee support for any particular website or source package.
